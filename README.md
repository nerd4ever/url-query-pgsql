## Installation

```bash
composer require nerd4ever/url-query-pgsql
```

## Usage

```php
use Nerd4ever\UrlQuery\Sql\UrlQueryPgSQL\UrlQueryPgSQL;

$urlQueryPgSQL = new UrlQueryPgSQL();

```
